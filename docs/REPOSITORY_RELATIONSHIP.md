# Repository Relationship

## Repository Role

`climateos-book` is the official ClimateOS Living Books publication system.

It hosts the public reading website and book source structure for ClimateOS-related books, including `Torch and Horizon / 远方与火炬`.

## Position in the Wider System

```text
ClimateOS Knowledge and Philosophy
        ↓
ClimateOS Living Books
        ↓
Public reading, multilingual publication and long-term knowledge communication
```

## Relationship to Other Repositories

- `Simon` is the personal headquarters and project index.
- `climateos-knowledge-garden` stores raw CRP, radar and internal working notes before publication.
- `eco-agent-system` contains environmental intelligence and decision-support framework logic.
- `BuildingOS` contains infrastructure governance platform logic.
- `climate-leadership` may become a shorter-form public communication site.

## Boundary

This repository should focus on:

- book publication
- public reading pages
- multilingual book structure
- PDF and manuscript publication workflow
- reader feedback and contribution routes

Do not use this repository as the main home for:

- raw personal notes
- weekly CRP harvests
- generic code experiments
- private collaboration records
- project management archives unrelated to publication

## Operating Rule

ClimateOS Book is the library and publishing house.

Only publication-ready or publication-supporting materials should enter this repository.
