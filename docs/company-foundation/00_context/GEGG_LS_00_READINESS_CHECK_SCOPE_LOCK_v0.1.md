## 1. Purpose

This document locks the scope for the GEGG Company Foundation Living Site v0.1.

The purpose of the site is to create a conservative, reviewable, public-facing draft narrative for GEGG’s emerging identity, mission, ethical boundary, theory base, project geography, and trust architecture.

This site is not a final corporate website. It is not an investor deck. It is not a fundraising page. It is not a RWA, tokenization, ecological asset finance, or financial-return promotion page.

It is a founder-review draft for organizing GEGG’s company foundation narrative before any public release.
